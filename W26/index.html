PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I
    Foundation in Calculus and algebra.
  • Engineering Physics
    Concepts of physics and thier application.
  • Basic Electronics
    Electronics circuits and components of it.
  • Programming in C
    Introduction to programming concepts and C language.

Second Year (SE)

  • Data Structures
    Introduction to Data structures and algorithms.
  • Object-Oriented Programming
    Concepts of object-oriented programming.
  • Processor Architecture
    Concepts of microcontroller and embedded systems.
  • Discrete Mathematics
    Introduction to discrete mathematical concepts.

Third Year (TE)

  • Operating Systems
    Introduction to operating systems and Linux OS.
  • Computer Networks and security
    Concepts of computer networks and their applications.
  • Cloud Computing
    Introduction to cloud computing concepts and technologies.
  • Data Algorithm and Analysis
    Introduction to Advanced Data structure and algorithms.

Fourth Year (BE)

  • Artificial Intelligence
    Introduction to AI concepts and applications.
  • Machine Learning
    Supervised and unsupervised learning techniques.
  • Cloud Computing
    Distributed systems and cloud platforms.
  • Project Work
    Capstone project in chosen specialization.
